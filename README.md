# Prácticas de Redes – Ejercicios 2 y 3

Repositorio con las actividades 2 y 3 de la práctica de redes, desarrolladas en **Python** con **Tkinter**.

---

## Ejercicio 2 – SMTP (Mailtrap)

Aplicación gráfica para el envío de correos electrónicos a un servidor SMTP de pruebas (Mailtrap).

- Conexión SMTP mediante credenciales
- Envío de correos con asunto y cuerpo HTML
- Visualización de respuestas y errores del servidor
- STARTTLS activado automáticamente si el servidor lo requiere

---

## Ejercicio 3 – NetworkLab (DNS y NTP)

Aplicación gráfica para trabajar con los protocolos DNS y NTP.

- DNS:
  - Resolución dominio → IP
  - Resolución IP → dominio (reverse DNS)
- NTP:
  - Consulta de la hora oficial a un servidor NTP público
